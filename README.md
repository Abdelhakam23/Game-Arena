# 🎮 GameArena — The Ultimate Gaming Experience

> A fully responsive gaming platform landing page built with pure HTML, CSS & Bootstrap — designed for competitive eSports communities.

![GameArena Banner](images/hero-img.png)

---

## 📌 Overview

**GameArena** is a modern, dark-themed gaming website landing page that serves as the face of a competitive eSports platform. It showcases games, tournaments, team members, and a contact form — all wrapped in a sleek neon-green-on-dark aesthetic.

---

## ✨ Features

- 🧭 **Sticky Navbar** with smooth scroll-spy highlighting active sections
- 🦸 **Hero Section** with an animated CTA, parallax background, and live player count badge
- 🎮 **Our Games Carousel** — responsive slider with hover overlays and genre tags
- ⚔️ **What We Do** — feature cards highlighting platform capabilities
- 🏆 **Tournaments Section** — live match cards and upcoming event listings
- 🧑‍🤝‍🧑 **Our Team** — player cards with a grayscale-to-color hover effect
- 🤝 **Sponsors Marquee** — auto-scrolling logo strip (left & right)
- 📬 **Contact Us** — full form with validation + community info panel
- 🦶 **Footer** — multi-column links, social icons, and legal links
- 📱 **Fully Responsive** — mobile, tablet & desktop support via Bootstrap grid

## 🚀 Getting Started

No build tools or dependencies needed. Just open and run.

### 1. Clone the Repository

```bash
git clone https://github.com/Abdelhakam23/game-arena.git
cd game-arena
```

### 2. Open in Browser

```bash
# Option A — just double-click index.html
# Option B — use VS Code Live Server extension (recommended)
```

> ✅ No npm, no bundler, no server required.

## 📱 Responsive Breakpoints

| Breakpoint | Layout |
|---|---|
| `< 576px` | Single column, stacked nav |
| `576px – 768px` | 2-column cards, inline buttons |
| `768px – 992px` | 2-column games, side-by-side sections |
| `≥ 992px` | Full desktop layout, 3-column games |
